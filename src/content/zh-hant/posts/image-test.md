## 功能說明

<Alert type="info" title="使用方法">

- **點擊圖片**：點擊任意圖片即可在燈箱中放大查看
- **關閉燈箱**：可以通過以下方式關閉：
  - 點擊右上角的 × 按鈕
  - 點擊圖片外的背景區域
  - 按下鍵盤的 ESC 鍵
- **圖片懶載入**：所有圖片都啟用了懶載入，提升頁面效能

</Alert>

## 圖片樣式特性

所有 Markdown 中的圖片都具有以下特性：

1. **自適應大小**：圖片會自動適應容器寬度
2. **圓角設計**：圖片帶有圓角，更加美觀
3. **陰影效果**：圖片帶有輕微陰影，增強層次感
4. **懸停效果**：滑鼠懸停時圖片會輕微上浮，提示可點擊
5. **點擊放大**：點擊後在燈箱中全屏查看高清大圖
6. **主題適配**：燈箱樣式適配亮色和暗色主題

## 效能優化

<Alert type="success" title="效能優化">

- 圖片採用懶載入技術，只在進入視口時才載入
- 燈箱使用 CSS 動畫，流暢且效能友好
- 移動端做了特別優化，確保良好的觸摸體驗

</Alert>
